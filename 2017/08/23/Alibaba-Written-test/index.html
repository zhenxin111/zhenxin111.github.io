CSS选择器优先级

权重记忆口诀：

从0开始，一个行内样式+1000，一个id+100，一个属性选择器/class或者伪类+10，一个元素名或者伪元素+1.

引用司徒正美文章里的权重表格：

选择器 表达式或示例 说明 权重
ID选择器 #aaa / 100
类选择器 .aaa / 10
标签选择器 h1 元素的tagName 1
属性选择器 [title] / 10
相邻选择器 selecter + selecter 拆分为两个选择器再计算 /
兄长选择器 selecter ~ selecter 拆分为两个选择器再计算 /
亲子选择器 selecter > selecter 拆分为两个选择器再计算 /
后代选择器 selecter selecter 拆分为两个选择器再计算 /
通配符选择器 * / 0
各种伪类选择器 如:link， :visited， :hover， :active， :target， :root， :not等 / 10
各种伪元素 如::first-letter,::first-line,::after,::before,::selection / 1

参考引用文章：

  • CSS Specificity: Things You Should Know By Vitaly Friedman
  • CSS选择器的权重详解-司徒正美

实现自定义事件机制：

实现如下方法：

  • on, 添加事件，可以给某个事件添加多个处理函数，这些函数可以被多次触发
  • once,添加一次性事件，在触发过一次后不再触发
  • fire，触发特定事件
  • off,移除特定的事件处理函数
