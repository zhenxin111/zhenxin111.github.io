树和背影

发表于 2017-12-21 | 分类于 生活 ， 绘画 | | 阅读次数

大概是从十月份开始画这幅，可这几个月来的秋招以及毕业论文的撰写压着我无法松懈。

等到论文送审，就出门看了《至爱梵高：星空之谜》油画电影，油画炽热的色彩在银幕中燃烧着，被画中饱满惊人的光热与颓唐寂寞的画家梵高间产生的落差，击中红心泪流满面。

影院回来，就觉得我得在画架前待一会儿，我想通过这样可能粗劣但是我所能做的方式去贴近一个落魄的天才。如果他知道世人有多么爱他的画、听到他们哪怕万分之一的赞叹，那该有多好。

以上是电影原声歌曲《Starry Starry Night》

至于我的涂鸦，其实无足轻重，只是我喜欢并且在做的一件事而已。
