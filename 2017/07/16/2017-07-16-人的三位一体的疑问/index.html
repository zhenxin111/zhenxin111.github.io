人的三位一体

人实际上可以看作一个三位一体，一个预先被埋藏在你身体里的野兽 + 一个活在故事中的自我 + 一个在空中俯视的看客，这才是完整的你。你们有没有体会过这种心情，就是你可以公允的同意一种对你自己客观不利的政策，你可以抛开自己的切身利益，去和人探讨某些东西。这个你，就是那个“空中的看客”我们的整个科学，大部分法理，都是来自空中的“看客们”。

疑问：
分裂？还是有倾向性后的整合？
野兽和看客的对立，是否有互通协调的部分？意味着人是被拉扯做路线还是可以设计规划？
有无对错之分？还是为了整体的生存舒适？生存重要还是舒适重要？
